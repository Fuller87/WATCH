# W.A.T.C.H.
**Warzone Advisor for Tactics, Crises & Heroes**

Ein Roster-Builder und Analyse-Tool für **Marvel Crisis Protocol**.  
Läuft komplett im Browser — keine Installation, kein Backend, keine Accounts.

---

## Features

- **Roster-Builder** — Charaktere hinzufügen, Typ & Anführer festlegen, Threat-Summe tracken
- **Roster-Bibliothek** — bis zu 10 Roster lokal speichern und laden
- **Text-Import** — Roster direkt aus jarvis-protocol.com per Plain-Text einfügen
- **Team-Analyse** — automatische Erkennung des Team-Stils (Allround / Attrition / Mission Play / Control)
- **Power-Budget** — Einnahmen vs. Kosten pro Charakter und für das gesamte Roster
- **Team-Empfehlung** — bestes 10-Char-Team für eine gewählte Crisis zusammenstellen
- **Crisis-Empfehlung** — passendste Crisis für dein Roster vorschlagen
- **Taktikkarten-Vorschläge** — passende Tactics Cards zum Team
- **Würfelrechner** — Angriffs- und Verteidigungswürfel simulieren
- **Roster-Vergleich** — zwei Roster gegenüberstellen (Schadensprofil, Range, Conditions, Warzone Control, Defensiv-Profil)
- **Charakter-Datenbank** — alle Chars durchsuchen mit Detailansicht

---

## Starten

Einfach `index.html` im Browser öffnen — oder die gehostete Version aufrufen und auf **"Zum Homescreen hinzufügen"** tippen.

---

## Offline-Nutzung

Die App ist eine Progressive Web App (PWA). Nach dem ersten Laden funktioniert sie vollständig ohne Internetverbindung.

---

## Daten

Alle Roster werden lokal im Browser gespeichert (`localStorage`). Es werden keine Daten an einen Server übertragen.
